# ilerimikroislemci
Tiva TM4C123GH6PM mikrodenetleyici ile gerçekleştirilmiş çeşitli projeleri içermektedir.
 Projeler, Code Composer Studio (CCS) kullanılarak geliştirilmiş olup, temel donanım kontrolü, LCD ekran yönetimi, UART haberleşmesi ve zamanlayıcı kullanımı gibi farklı konuları kapsamaktadır.

# Proje İçeriği

1. Temel LED Kontrolü 

2. Seri Port İletişimi

3. LCD Ekran Kullanımı 

4. Zamanlayıcı Tabanlı İşlemler

5. Matematiksel İşlemler ve Floating Point Kullanımı

6. Gelişmiş Buton Kontrolü

7. Saat Gösterimi ve Zamanlama

Gibi çeşitli projelerin yer aldığı bir klasördür.

# Gereksinimler

Projelerin çalıştırılabilmesi için gerekli araçlar:
- Tiva TM4C123GH6PM geliştirme kartı
- Code Composer Studio (CCS) v12 veya üzeri
- 16x2 LCD ekran (bazı projelerde kullanılmıştır)
- Seri port haberleşmesi için bir terminal uygulaması (örneğin, PuTTY)
